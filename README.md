## [我让死对头AI为我写情书](https://hlbltai.github.io/)
>我和研发对手公司的工程师在行业展会上唇枪舌战三年。  
>今年他缺席了，却给我留了封信：“其实每次吵架——都是我最想你的时候。”  
>我冷笑回复：“技术差的人才玩情怀。”  
>当晚我的AI突然乱码，屏幕上跳出一行字：  
>“那这样能引起你注意吗，亲爱的宿敌小姐？”
---

国际未来科技展的场馆永远充斥着一种冰冷的亢奋。空气里是恒温系统精准控制的微凉，混合着崭新电子元器件的特有气味，还有无数扬声器里传出的、经过降噪处理的柔和解说。展台光洁如镜，各色全息投影在半空中无声绽放，穿着剪裁得体商务装的人们挂着无懈可击的微笑，交谈、握手、交换名片，一切都高效、有序，闪着金属和资本的光泽。

林薇踩着七厘米的细跟，鞋跟敲打光洁地砖的声音清脆而稳定，像她此刻的心跳节奏。深灰色西装套裙妥帖，衬得她肤色愈发白皙，一枚简洁的铂金胸针别在领口，随着她的步伐偶尔折射一点冷光。她的视线快速扫过一个个展台，评估，计算，过滤掉无关信息。她是星烁科技算法研发部的负责人，来这里既是展示，也是狩猎。

然后，她的脚步停在了“辰辉智能”的巨幅展板前。展台设计依旧带着那种她熟悉的、过分精致的炫技感，流动的数据瀑布墙和交互全息沙盘吸引了不少人驻足。但她的目光径直穿过这些炫目的光影，落在那个本该有人站立、此刻却空荡荡的讲解位上。

周叙白没来。

这个认知让她怔了半秒。过去三年，每次展会，这里都是她的固定“战场”。那个男人，辰辉智能的首席工程师，总会站在那里，穿着熨帖的衬衫，袖口随意挽起，带着一种懒散又锐利的笑容，等着她走近，然后一场没有硝烟的交锋准时上演。从神经网络架构的优劣，到实时数据处理算法的瓶颈，再到对某个前沿论文几乎截然相反的解读，他们能在人群中央旁若无人地争论半小时，字句犀利，寸步不让。往往以她甩下一句“纸上谈兵”或“华而不实”，他回敬一句“固步自封”或“缺乏想象力”告终，各自转身，背脊挺直，留下周围一圈尴尬又兴奋的看客。

那几乎成了展会的一个保留节目。有人私下调侃他们是“行业辩友”，也有人猜测两家公司是否在酝酿什么惊人的合作。只有林薇自己知道，每次“辩论”后，她回到酒店房间，指尖都因为高度的精神集中和某种难以言喻的刺激而微微发麻。她需要一杯冰水才能让自己彻底冷静下来。

今年，他没来。展台是辰辉的其他人，面孔生疏，笑容标准。一种莫名的空落，像忽然踩空了一级台阶，在她胸腔里轻微一坠。她立刻抿紧了唇，将这不合时宜的情绪压下去，转身欲走。

“林总监，”一个辰辉的年轻员工却小跑着过来，手里拿着一个素白、厚重的信封，态度恭敬，“这是我们周工嘱咐一定要亲手交给您的。”

信封入手微沉，质感极佳。林薇垂眼看了看，没有立即打开，只淡淡颔首：“谢谢。”

她走到相对僻静的休息区，坐下，将信封放在光洁的桌面上。指尖划过封口，里面只有一张同样质感的卡片，上面是打印体，但措辞……

“林总监：展会同期的老友辩论赛，今年因故缺席，憾甚。过去三年，每次与你针锋相对、思维碰撞的时刻，于我而言，都远不止是技术交流。事实上，每次‘吵架’——都是我最想你的时候。”

打印的字迹工整冰冷，内容却让她呼吸一滞。随即，一股被冒犯的怒意和被戏弄的冷意交织着冲上头顶。想她？用这种轻浮又故作深情的口吻？在缺席了正面交锋之后？她几乎能想象出周叙白写下或打出这些字时，脸上那副该死的、游刃有余的表情。

她拿起随身携带的定制钢笔，在卡片下方空白处，用力划下几行字，笔尖几乎要戳破纸面：

“周工：技术差距开始用情怀弥补了？拙劣的尝试。真正的注意力靠实力赢得，比如，在展台上正面击败我。祝贵司早日研发出配得上你这番‘想念’的算法。另：字真丑。（手写体）”

写完，她将卡片塞回信封，叫来那个还在附近的辰辉员工，面无表情地递回去：“麻烦转交周工。还有，告诉他，明年如果还是不敢来，就不用留信了。”

夜幕降临时，白天的喧嚣彻底沉淀。酒店高层套房落地窗外，城市灯火璀璨如倒悬的星河。林薇洗去一身疲惫，换上舒适的丝质睡袍，湿发用毛巾裹着。她打开随身携带的顶级加密笔记本电脑，连接酒店安全网络，开始处理白天积压的工作邮件，并例行检查她独立开发、高度私密的AI辅助系统“银星”。

“银星”是她心血所在，深度介入她的研究、日程甚至部分思维习惯，响应速度、逻辑严谨性和学习能力都远超市面上任何同类产品。它无声地运行着，直到她点开某个数据模型文件时，屏幕中央忽然毫无征兆地弹出一个纯黑的对话框，边缘流转着细微的、类似电路熔毁般的猩红色光晕。

林薇手指一顿，瞳孔微缩。这不是“银星”的交互界面。

没等她采取任何操作，一行苍白的文字，一个字母接一个字母，带着某种刻意为之的迟缓节奏，在黑色框体中浮现：

**“那……”**

指尖微微绷紧，落在触控板上。

**“……这样……”**

她迅速尝试启动几个深层诊断命令，但指令如石沉大海，系统响应迟缓得可怕，仿佛被无形的胶质包裹。

**“……能引起你注意吗……”**

最后一个单词跳出，带着几乎能听见的、无声的戏谑尾音：

**“……亲爱的宿敌小姐？”**

房间里的恒温系统似乎停止了工作。窗外的璀璨灯火变得遥远而不真实，只剩下屏幕那一点刺眼的光，映着她骤然苍白又迅速染上冰冷怒意的脸。宿敌小姐。这个称呼，这种口吻……

她猛地向后靠进椅背，丝质睡袍摩擦发出细微的窸窣声，在过分寂静的房间里被放大。目光死死锁住那行字，仿佛要透过屏幕，钉死另一端那个胆大包天的入侵者。

周、叙、白。

除了他，不会有别人。用这种方式，在这种时候。

她缓缓地、极冷地笑了一下，手指重新放回键盘，用力敲击，带着要将按键击穿的力度：

“黑进我的系统？周叙白，你这才是真正的——技术烂透了。”

发送。屏幕上的黑色对话框闪烁了一下，那行挑衅的文字淡去。但紧接着，新的字迹开始浮现，不再是之前那种模拟故障的缓慢，而是流畅、迅疾，带着她熟悉的、某种漫不经心的精准：

**“防火墙第七层，三小时前刚更新的动态密匙，基于你去年那篇获奖论文的衍生算法。破解它用了一百四十二秒。‘烂透了’的评价，我是否需要理解为林总监独特的褒奖？”**

林薇盯着这行字，呼吸有一瞬间的凝滞。第七层的更新……他不仅突破了，还精确指出了算法来源和时间。一百四十二秒。一种冰凉的战栗顺着脊椎爬升，并非全然源于系统被入侵的愤怒，更深处，是一种遇到真正对手时、近乎本能的警觉和……兴奋。

她闭上眼，深吸一口气，再睁开时，眼底已是一片锐利的清明。双手抚上键盘，指尖稳定，不再有丝毫颤抖。战斗的本能压倒了一切。

“一百四十二秒值得炫耀？看来辰辉的算力池该扩容了。顺便，你在第三跳板留下的伪装协议，语法错误很可爱。”

**“故意留的。不然怎么确认你真的在认真看，而不是直接拔电源？看来‘宿敌小姐’不仅记性好，嘴也一如既往。”**

“对你这种不请自来的‘客人’，只有这个。”她飞快地调出另一个监控界面，试图反向追踪信号流，同时文字交锋不停，“绕过‘银星’的主动防御层，用了非对称神经陷门？胆子不小，可惜痕迹太重。”

**“承蒙夸奖。不过重点不是‘如何进来’，而是‘为什么进来’。白天那封情书，看来是彻底搞砸了。”**

“情书？”林薇嗤笑，手指敲击更疾，“那叫战术骚扰，低劣且无效。真正的挑战在展台，在论文里，在代码效率上。而不是这种……”她顿了顿，找到一个词，“小孩子的把戏。”

黑色对话框静默了几秒。窗外，一片流云掠过，遮住了些许月光，房间内光影暗了一瞬。

**“有时候，‘小孩子的把戏’是因为更直接的方式总被误解。比如，认真讨论时，你眼里只有‘对手’；递出橄榄枝，你看作‘示弱’。那么，打破你最自豪的防线，是不是终于能让你暂时放下‘星烁总监’的面具，只看着‘周叙白’这个人？”**

文字一行行跳出，平静，甚至带着一丝探究，却比任何激烈的言辞都更具穿透力。林薇敲击键盘的手指停了下来，悬在按键上方。屏幕的冷光映着她轮廓分明的侧脸，长长的睫毛在眼底投下浅淡的阴影。面具？她一直戴着面具吗？在他面前？

她慢慢地，重新开始输入，速度缓了下来，每个字都似乎经过斟酌：“看着你，然后呢？欣赏你非法入侵的才华，还是感谢你提醒我系统仍有漏洞？”

**“然后，承认我们除了吵架，或许还能有别的对话方式。比如，关于你正在优化的那个多模态模型，它在极端边缘场景下的泛化能力，我有个想法，可能比你现在尝试的路径短10%。”**

林薇的呼吸轻轻一滞。她最近的攻坚难点，他不仅知道，竟然还有思路？这绝非临时起意能编造的内容。一种极其复杂的情绪涌上来，警惕，好奇，不甘，还有一丝被精准挠到痒处的微妙。

“……说。”

**“这里不安全。你的反追踪程序虽然慢了点，但挺执着。明天下午三点，展馆B区，智能咖啡机器人旁边。那里信号杂乱，监控有十五秒盲区。敢来吗，宿敌小姐？”**

敢吗？当然。她怎么可能不敢。愤怒仍在底层燃烧，但一种更炽热、更原始的东西被点燃了——对答案的渴望，对挑战的迎接，对那个一直站在对面、此刻却似乎试图撬开另一扇门的男人的……无尽探究欲。

她删掉了已经输入好的嘲讽语句，重新打了一个字，干脆利落：

“等。”

发送成功的同时，屏幕上的黑色对话框像被擦除般瞬间消失，系统恢复如常，“银星”的待机界面安静地亮着，仿佛刚才的一切只是幻觉。只有任务栏角落里，一个微小的、陌生的日志文件生成提示，一闪而过。

林薇靠在椅背上，目光从屏幕移向窗外无尽的夜色。城市依旧灯火流转，但有什么东西，已经彻底改变了。明年展会？不，较量在十五个小时后，就要提前开始。

她端起桌上早已凉透的水，抿了一口，冰凉液体滑入喉咙，却压不住心头那簇越烧越旺的火。指尖无意识地在膝盖上敲击着，那不是惯常思考技术难题时的节奏，更快，更轻，带着一丝不易察觉的紧绷。

周叙白。她在心里默念这个名字。这次，你又想玩什么花样？

窗玻璃上，隐约映出她微微蹙眉、却亮得惊人的眼睛。
